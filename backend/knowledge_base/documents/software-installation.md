# Software Installation Guide

## Overview

All software installations must go through the Company Software Portal to ensure license compliance and security standards. Unapproved software installations are flagged by endpoint protection and may be removed automatically.

## Using the Company Software Portal

1. Open the **Company Portal** app on your workstation (pre-installed on all managed devices).
2. Browse the catalog or use the search bar to find the application you need.
3. Click **Install** and wait for the download and installation to complete.
4. Some applications require a restart — save your work before proceeding.

Available categories include: Productivity, Development Tools, Design, Communication, and Utilities.

## Requesting Software Not in the Portal

1. Go to **ServiceNow > Service Catalog > Software Request**.
2. Fill in the application name, version, business justification, and number of licenses needed.
3. Submit the request. It routes to your manager for approval, then to IT Security for review.
4. Typical turnaround is 3-5 business days. High-risk or unlicensed software may take longer.
5. Once approved, the software appears in your Company Portal under **Assigned Apps**.

## Administrative Rights

- Standard users do not have local admin rights on corporate devices.
- If an installation requires admin privileges, submit a **Privileged Access Request** in ServiceNow.
- Temporary admin access is granted for 4-hour windows and logged for audit purposes.
- Developers may request a standing admin exception through their engineering manager.

## License Management

- All commercial software is managed through a central license pool.
- When you no longer need an application, uninstall it via the Company Portal to release the license.
- License audits run quarterly. You may receive a notification to confirm you still need assigned software.
- Open-source tools must still be vetted by IT Security before installation to ensure no licensing conflicts.

## Common Issues

- **Installation stuck at "Downloading":** Check your network connection and ensure VPN is active if working remotely. Restart the Company Portal app.
- **"Insufficient permissions" error:** You need a Privileged Access Request. See the Administrative Rights section above.
- **Application crashes after install:** Verify system requirements on the software's catalog page. Run Windows Update or macOS Software Update to ensure OS patches are current.
- **Software removed unexpectedly:** Endpoint protection may have flagged it as unapproved. Check your email for a notification from IT Security and submit a request if the software is needed.
